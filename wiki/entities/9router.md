# 9router

9router là một dịch vụ cung cấp API AI giúp người dùng truy cập các mô hình mã nguồn mở (Open Source Models) hoàn toàn miễn phí.

## Đặc điểm
- **Chi phí 0đ:** Phù hợp cho mục đích giáo dục, thử nghiệm và cài đặt các agent như **[[OpenClaw]]**.
- **Kết nối:** Cung cấp hạ tầng để gọi các model AI mạnh mẽ mà không cần phải trả phí trực tiếp.

## Liên kết
- [[OpenClaw]] — Một agent phổ biến sử dụng 9router để đạt chi phí thấp nhất.
- [[Inner-Harness]] — 9router cung cấp model cần thiết cho Inner Harness hoạt động.
