# Claude Code

Công cụ dòng lệnh (CLI) của Anthropic, được thiết kế như một AI agent mạnh mẽ chuyên biệt cho việc lập trình.

## Đặc điểm
- **Inner Harness Chuẩn mực:** Được cộng đồng đánh giá là ví dụ tiên phong và tốt nhất hiện nay về thiết kế **[[Inner-Harness]]** (cách tổ chức context, nạp system prompt, v.v.).
- **Tích hợp sâu:** Có khả năng thực thi code, chạy test, sửa lỗi trực tiếp trong môi trường dự án.
- **Tính tự trị cao:** Có khả năng thực hiện các nhiệm vụ phức tạp từ việc viết mã đến tối ưu hóa cấu trúc dự án.

## Liên kết
- [[Inner-Harness]] — Claude Code là minh chứng thực tế cho khái niệm này.
- [[Gemini-CLI]] — Một công cụ CLI tương tự nhưng dựa trên mô hình Gemini.
- [[Outer-Harness]] — Claude Code cần lớp Outer Harness của doanh nghiệp để quản lý chi phí và quy trình nộp code.
